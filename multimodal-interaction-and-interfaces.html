Multimodal interaction and interfaces

Exploring

Role: Concept developer, prototype

Partners: Gustav Rannestig

Scope: 6 weeks, halftime

ABOUT THE PROJECT

The project is an application developed using the Xbox Kinect controller to create a virtual environment where an user can interact with a virtual ball in two-dimensional space. The virtual environment consists of a visual representation of the userÂ´s body as a stick-figure, the ball and an auditory display that renders sound based on the interaction the user makes with the virtual ball. The project was conducted to test what impact sound had on the interaction with virtual objects and was tested on three test subjects. The results concludes that for the sound feedback to be able to give the feeling or illusion of haptics on its own, the virtual world should be built with different constraints and limitations than ours. We created a virtual world where the visual aspect was required for the user to be able to understand it. A video-demonstration of the project can be found at https://vimeo.com/56747207
